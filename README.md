# 🧠 Brain_Tumor_detection-CNN-Pytorch

A Deep Learning project that classifies brain tumors from MRI scans into four categories: Glioma, Meningioma, Pituitary, and No Tumor.

---

## 👤 Creator
**Suraj Santra**

---

## 📊 Dataset

- Brain Tumor MRI Dataset (Kaggle)

**Classes:**
- Glioma  
- Meningioma  
- Pituitary  
- No Tumor  

---

## 🚀 Model Performance

- Model: ResNet-18 (Pretrained & Fine-tuned)  
- Training Accuracy: ~92–95%  
- Testing Accuracy: ~85–90%  
- Epochs: 20  
- Optimizer: Adam  
- Scheduler: StepLR
